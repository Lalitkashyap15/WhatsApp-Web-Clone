# WhatsApp-Web-Clone

A real-time WhatsApp-inspired web chat application built with Firebase and JavaScript. Features include authentication, one-to-one messaging, file sharing, emoji support, profile management, message pinning, blocking users, and Jitsi-powered video calling.
# WhatsApp Web Clone

A real-time chat application inspired by WhatsApp Web, built using Firebase and JavaScript.

## Features

- User Authentication
- Real-Time Messaging
- File & Image Sharing
- Emoji Support
- User Profiles
- Online/Offline Status
- Message Pinning
- Message Deletion
- Block Users
- Video Calling using Jitsi Meet
- Firebase Cloud Storage Integration

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase Authentication
- Firestore Database
- Firebase Storage
- Jitsi Meet API
